About Alfe AI

Alfe AI is an open source platform that combines AI-assisted image design, software development, and task management.

All source code is available on GitHub for self-hosting or customization. Alfe AI has a privacy-first philosophy, and your data is only used for purposes required by the application. Data is not sold or used for advertising.
